## API Documentation

### Tools

| Tool | Description |
|------|-------------|
| `graphrag_global_search` | Community-aware global search across all indexed content |
| `graphrag_local_search` | Entity-focused local search with graph traversal |
| `graphrag_drift_search` | Dynamic reasoning with iterative follow-up queries |
| `graphrag_basic_search` | Direct vector similarity search |
| `graphrag_build_index` | Build or update the knowledge graph index |
| `graphrag_get_statistics` | Get index statistics and health information |

### Prompts

| Prompt | Description |
|--------|-------------|
| `graphrag_explore` | Exploratory knowledge graph analysis prompt |
| `graphrag_analyze` | Deep analysis prompt for specific topics |
| `graphrag_summarize` | Summarization prompt for indexed content |
